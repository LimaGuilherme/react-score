# React Score

This package have files with basic example of score board.

## Getting Started

```
$ git clone https://github.com/LimaGuilherme/react-score.git
```
```
$ cd react-series
```
```
$ npm install
```
```
$ npm install -g webpack
```
```
$ npm run dev
```
http://localhost:8080/
